INTRODUCTION AND GAMEPLAY

This is a browser-based space survival game where players control a rocket and navigate through dangerous regions of space.
Your mission is to pilot a rocket through space while avoiding incoming asteroids and planets. The rocket can be controlled using either the WASD keys or the Arrow Keys.
Players start with 3 lives. Colliding with an asteroid or planet causes the loss of a life. 



LEVELS

The game consists of 3 levels , of which the difficulty increases as you progress - 
Level 1: Short distance to destination
Level 2: Increased travel distance 
Level 3: Longest distance and ultimate survival challenge
